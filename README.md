# 🎬 IMDb Movie Review Sentiment Analysis

## 📌 Overview
This project classifies movie reviews as **positive or negative** using Natural Language Processing techniques.

## 🚀 Features
- Web scraping of IMDb reviews using BeautifulSoup  
- Text preprocessing and cleaning  
- TF-IDF vectorization  
- Sentiment classification using Logistic Regression and Naive Bayes  
- Deployment using Gradio on Hugging Face  

## 📊 Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- NLTK  
- Gradio  

## 📈 Model Performance
- Evaluated using accuracy, F1-score, and confusion matrix  

## 🔗 Live Demo
https://huggingface.co/spaces/divyashekar/imdb-sentiment-analysis  

## 📂 Dataset
- Scraped IMDb reviews dataset (CSV format) for initial data collection  
- Full IMDb dataset sourced from Kaggle for model training (link provided in data folder)  

## ▶️ How to Run
1. Clone the repository  
2. Install dependencies  
3. Run the notebook  

## 📌 Result
The model successfully classifies user input reviews into positive or negative sentiment.
